



# [Trumpet fingering tool app](https://nicolasloizeau.github.io/trumpet-fingering/)

A web app to help learn trumpet fingerings.

Try it online: [https://nicolasloizeau.github.io/trumpet-fingering/](https://nicolasloizeau.github.io/trumpet-fingering/)


Build and run the JS app locally:
```bash
npx webpack

```
